Projekt

Panel stworzony dla Państwowej Wyższej Szkoły Zawodowej w Chełmie w celu przechowywania danych dot. sprzętu w poszczególnych salach, a także informacji przedstawiających wszelkie zmiany techniczne.

Baza danych

Struktura - items

id - mediumint(8),
nazwa - limit 25 znaków,
historia - limit 500 znaków,
pomieszczenie - limit 20 znaków,
data - typ date,
created_by - przechowuje id twórcy,
created_at - data stworzenia,
updated by - ID osoby aktualizującej,
updated at - ostatnia aktualizacja.

Struktura - admin_accounts

id - mediumint(8),
user_name - limit 50 znaków,
password - limit 255 znaków,
series_id - limit 60 znaków,
token - 255 znaków,
expires - data,
admin_type - limit 10 znaków.

Zarządzanie

Opis sposobu zarządzania panelem.

Dodawanie sprzętu

Sprzęt -> Lista -> Dodaj nowe -> wprowadzenie odpowiednich wartości -> zapis.

Edycja i usuwanie sprzętu

Sprzęt -> Lista -> niebieska ikonka w zarządzaniu.

Tworzenie użytkowników

Administracja -> Dodaj -> Wprowadzenie Loginu & Hasła -> wybór typu konta -> zapis.

Nadawanie uprawnień

Administracja -> niebieska ikona -> edycja wedle uznania. Usunięcie przy użyciu czerwonej ikony.
